# SaaS Dashboard Platform

Production-style multi-tenant SaaS dashboard platform built with:

- Next.js
- TypeScript
- Node.js
- PostgreSQL
- Redis
- Docker
- CI/CD
- AWS/GCP deployment

## Features

- JWT Authentication
- RBAC Authorization
- Analytics Dashboard
- API Rate Limiting
- File Uploads
- Subscription Billing
- Dockerized Infrastructure
- CI/CD Pipeline
